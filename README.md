# FootballManager
FootballManager, C++

## FOOTBALL MANAGER
## 최적전술매니저


### 프로그램 내 알고리즘을 통해 사용자의 신체능력(수치화된)과 선호하는 위치, 주발등을 입력하여
### 자신에게 최적화된 포지션을 우선순위에 따라 추천해주는 기능

### 선수데이터를 저장해두면 저장된 선수(사람)데이터를 이용해
### 선호하는 포메이션을 입력하면 보유 데이터중 가장 강한 스쿼드를 자동으로 완성해주는 기능

### 일반인, 축구동호회, 선수들 모두 목적에 맞게 사용가능
